# Sistema de Eventos em Java

Projeto desenvolvido para cadastro e gerenciamento de eventos.

## Funcionalidades
- Cadastro de eventos
- Listagem de eventos
- Participação em eventos
- Cancelamento de participação

## Tecnologias
- Java
- Programação Orientada a Objetos

## Como executar
Execute o arquivo Main.java em uma IDE ou no Replit.
